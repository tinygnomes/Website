# Overview

## Goals

1.  Provide a clear landing page for potential users.
2.  Showcase features (Calendar, Compliance, Tasks, etc.).
3.  Display testimonials and trust signals (client logos).
4.  Allow users to sign up or schedule a demo.
5.  The website MUST be fully responsive and handle both mobile and desktop devices.

## Technology Stack

The project has been migrated to a **Static Site** architecture with **AI-Assisted Literate Programming (ALP)**.

*   **Structure & Logic**: Defined in these Markdown files (ALP).
*   **Visuals**: Sourced from Figma -> Locofy exports (Plain CSS).
*   **Code Generation**: HTML is generated by the AI agent based on the Markdown specifications.
*   **No Build System**: The output is pure HTML/CSS/JS that runs directly in the browser.

### Directory Structure

*   `website/`: The public `public_html` folder.
    *   `index.html`: The main entry point.
    *   `css/`: Plain CSS files (migrated from Locofy).
    *   `assets/`: Images and static resources.
    *   `js/`: Custom JavaScript for interactions.
*   `Locofy/`: A read-only reference of the design export.

## Glossary

*   **ALP (AI-Assisted Literate Programming)**: The workflow used in this project where documentation is the source of truth.
*   **Locofy**: A tool used to export Figma designs into production-ready HTML/CSS.
*   **TinyGnomes**: The project management SaaS product this website promotes.
